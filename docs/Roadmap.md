## 로드맵

### 1. Auction 핵심 로직 구현
**구현 대상**
- `AuctionController`, `AuctionService`, `AuctionRepository`

**주요 기능**
- 경매 생성
- 입찰 처리
- 즉시 구매 로직
- 경매 종료 처리
- WebSocket 기반 실시간 입찰 기능

### 2. 테스트 코드 작성
