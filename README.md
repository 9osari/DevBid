### 프로젝트 명

**DevBid** (데브비드)

### 구상

개발자들을 위한 실시간 경매 플랫폼

### Backend

- **Java 21 LTS**
- **Spring Boot**
- **Spring Data JPA**
- **MySQL** (DB)
- **Redis** (캐시, 세션, 실시간 데이터)

### Frontend

- **Thymeleaf** (초기)
- **JavaScript** (실시간)
- **Bootstrap** (UI)

### 실시간 통신

- **WebSocket** (실시간 경매, 채팅)
- **STOMP** (메시징 프로토콜)

기능

- 회원가입(소셜) / 로그인(소셜)
- 게시판 / 검색

핵심기능

- 판매자 경매 생성(출품) / 구매자 품목 입찰
- 입찰 / 즉시구매

부가 기능(나중)

- 채팅?
- 즐겨찾기
- 구매후기
